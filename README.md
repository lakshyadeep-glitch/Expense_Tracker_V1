# Expense Tracker

A simple Python application that helps users manage and track their daily expenses.

## Features

* Add new expenses
* View all saved expenses
* Calculate total spending
* Save expenses to a text file
* Load saved expenses automatically when the program starts

## Technologies Used

* Python
* Lists
* Dictionaries
* Functions
* File Handling

## Project Structure

```text
Expense-Tracker/
│
├── main.py
├── expenses.txt
└── README.md
```

## How to Run

1. Install Python.
2. Clone or download this repository.
3. Open the project folder.
4. Run:

```bash
python main.py
```
