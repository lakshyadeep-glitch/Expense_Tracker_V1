# Expense Tracker

A simple command-line Expense Tracker built with Python. This application allows users to add expenses, view saved expenses, calculate the total amount spent, and store expense records in a text file.

## Features

* Add one or multiple expenses
* View all expenses added during the current session
* Calculate the total amount spent
* Save expenses to a text file (`expenses.txt`)
* Menu-driven interface

## Technologies Used

* Python
* Lists
* Dictionaries
* Functions
* Loops
* File Handling

## How to Run

1. Install Python 3.
2. Clone or download this repository.
3. Open the project folder.
4. Run:

```bash
python main.py
```
